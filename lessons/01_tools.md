# Урок 01 — Инструменты

> Перед тем как что-то строить, тебе нужны инструменты. На всё уйдёт ~2 часа.

## Стек

| Инструмент | Для чего | Действие |
|-----------|---------|---------|
| [Claude](https://claude.ai) | AI-ассистент: код, дебаг, тексты, планирование | Зарегайся (Pro $20/мес — worth it) |
| [Bolt.new](https://bolt.new) | Сборка приложений через текст — фронт + бэкенд + БД | Создай тестовый проект |
| [GitHub](https://github.com) | Хранение кода. Google Drive для кода + нужен для деплоя | Создай аккаунт |
| [Vercel](https://vercel.com) | Хостинг — берёт код с GitHub, выкладывает в интернет бесплатно | Зарегайся через GitHub |
| [Netlify](https://netlify.com) | Альтернатива Vercel | (выбери одно) |
| [Supabase](https://supabase.com) | Бесплатная БД: юзеры, данные, авторизация из коробки | Создай проект, сохрани URL + API key |
| [Polar.sh](https://polar.sh) | Приём оплаты: подписки, инвойсы, налоги | Создай аккаунт (продукты — позже) |
| [Lemon Squeezy](https://lemonsqueezy.com) | Альтернатива Polar | |
| [Namecheap](https://namecheap.com) / [Cloudflare](https://cloudflare.com) | Домен ~$10/год | Подожди пока есть идея |
| [0to1.sh](https://0to1.sh) | Карточка проекта, трекер прогресса, подключи GitHub | Зарегайся + создай карточку |

## Ключевые инсайты

- **Bolt.new** = главное оружие. Описываешь на русском/английском — получаешь приложение.
- **Supabase** уже содержит авторизацию (логин/регистрация для твоих юзеров) — не надо строить с нуля.
- **Домен** — не покупай пока нет чёткой идеи. Короткое + свободное = норм.
- **Vercel** чуть проще Netlify для старта.
