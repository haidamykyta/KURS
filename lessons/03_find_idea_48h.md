# Урок 03 — Найди прибыльную идею за 48 часов

> Не придумывай с нуля — реализуй исходя из рынка.

## Метод 1: Укради и сделай лучше

**Инструменты:** [TrustMRR](https://trustmrr.com), [IndieHackers/products](https://indiehackers.com/products), [Acquire.com](https://acquire.com)

**Шаги:**
1. Зайди на TrustMRR → фильтр MRR $1K–$15K
2. Открой 20 проектов
3. Для каждого: сайт → что делают → слабые места

**Что искать (слабые места):**
- Устаревший дизайн
- Нет мобильной версии
- Завышенная цена
- Недостающие фичи

**Пример:** Десятки email-тулов с $3-5K MRR, ужасный UX, $49/мес. → Ты делаешь аналог для инди-хакеров за $19/мес → забираешь часть рынка.

**Action:** Найди 5 проектов с $1-15K MRR, выпиши что можно сделать лучше.

## Метод 2: Reddit Mining

**Правило:** Не используй поиск Reddit — он плохой. Используй Google.

**Google-запросы:**
```
site:reddit.com/r/smallbusiness "looking for a tool"
site:reddit.com/r/SaaS "wish there was"
site:reddit.com "I'd pay for" app
site:reddit.com "someone should build"
```

**Сабреддиты с готовыми идеями (просто листай):**
- [r/AppIdeas](https://reddit.com/r/AppIdeas)
- [r/SomebodyMakeThis](https://reddit.com/r/SomebodyMakeThis)

**Action:** 1 час. 5 запросов через Google + полистать оба сабреддита. Выпиши 10 запросов/жалоб.

## Метод 3: Шпионь за конкурентами

**Инструменты:** [Product Hunt](https://producthunt.com), [AlternativeTo](https://alternativeto.net)

**На Product Hunt:**
- Открой интересную категорию → сортируй по недавним
- Ищи проекты с **50–200 аппвоутами** (не топ с тысячами — там большие компании)
- Читай комменты: "нет такой-то фичи", "дорого", "я бы использовал если бы..."

**На AlternativeTo:**
- Вбей известный продукт (Notion, Trello, Calendly)
- Смотри маленьких и недавних конкурентов

**Action:** 5 недавних проектов на PH → прочитай комменты → выпиши что людям не хватает.

## Метод 4: Автоматизируй чужую боль

**Инструменты:** [Upwork](https://upwork.com), [Fiverr](https://fiverr.com)

**Логика:** Повторяющаяся задача за $50–500 = SaaS за $29/мес.

**Категории для поиска:** Data Entry, Web Scraping, Virtual Assistant

**Примеры задач:**
- Создать отчёт из данных
- Перенести данные из X в Y
- Настроить автоматическую рассылку
- Почистить базу email
- Сделать скриншоты 100 страниц

**Пример:** Сотни заказов "scrape emails from LinkedIn" по $100-300. Apollo.io автоматизировал это → сейчас стоит $1.6B. Тебе не нужен Apollo — возьми одну задачу.

**Action:** Upwork → найди 5 задач которые заказывают регулярно.

## Фильтр: стоит ли строить?

5 вопросов, каждый "да" = 1 балл:

| # | Вопрос | Да/Нет |
|---|--------|--------|
| 1 | Люди уже платят за похожее? (погугли или спроси Claude) | |
| 2 | Могу сделать MVP за неделю в Bolt? | |
| 3 | Могу объяснить ценность в одном предложении? ("X для Y чтобы Z") | |
| 4 | Знаю где сидят мои клиенты? (сабреддит, чат, комьюнити) | |
| 5 | Клиенты — бизнесы, а не обычные люди? (B2B платит больше) | |

**Результат:**
- 4-5 баллов → **делай**
- 3 балла → подумай ещё
- 0-2 балла → следующая идея
